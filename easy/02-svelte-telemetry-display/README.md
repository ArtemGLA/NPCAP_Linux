# Задание 2: Telemetry Display

**Уровень:** Лёгкий  
**Технологии:** Svelte, TypeScript  
**Время выполнения:** 2-3 часа

## Описание

Наземные станции управления БПЛА отображают телеметрические данные в реальном времени. В этом задании вам нужно создать компонент для отображения основных параметров полёта.

## Цель

Создать Svelte компонент `TelemetryPanel`, который:

1. Отображает текущие данные телеметрии (высота, скорость, заряд батареи)
2. Показывает индикаторы состояния с цветовой кодировкой
3. Обновляется при изменении данных

## Что нужно реализовать

### 1. Компонент TelemetryPanel

Файл: `src/lib/TelemetryPanel.svelte`

```svelte
<script lang="ts">
  import type { TelemetryData } from './types';
  
  export let data: TelemetryData;
</script>

<!-- Реализуйте разметку -->
```

### 2. Типы данных

Файл: `src/lib/types.ts` (уже создан)

```typescript
export interface TelemetryData {
  altitude: number;        // метры
  speed: number;           // м/с
  battery: number;         // процент (0-100)
  lat: number;             // широта
  lon: number;             // долгота
  satellites: number;      // количество спутников
  mode: FlightMode;
  armed: boolean;
}

export type FlightMode = 'STABILIZE' | 'GUIDED' | 'AUTO' | 'LAND' | 'RTL';
```

### 3. Требования к UI

1. **Altitude (Высота)**
   - Отображать значение в метрах
   - Зелёный цвет если < 100м, жёлтый 100-200м, красный > 200м
   - Иконка стрелки вверх/вниз в зависимости от направления

2. **Speed (Скорость)**
   - Отображать значение в м/с
   - Индикатор направления (стрелка)

3. **Battery (Батарея)**
   - Прогресс-бар с процентом
   - Зелёный > 50%, жёлтый 20-50%, красный < 20%

4. **GPS**
   - Координаты с 4 знаками после запятой
   - Количество спутников с индикатором качества

5. **Status**
   - Режим полёта
   - Статус армирования (✓ или ✗)

## Подсказки

1. Используйте reactive statements (`$:`) для вычисления цветов
2. Для прогресс-бара используйте CSS width в процентах
3. Используйте CSS variables для цветовой схемы
4. Компоненты-индикаторы можно сделать переиспользуемыми
